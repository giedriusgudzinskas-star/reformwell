# Reformwell — exercise image prompts (photorealistic)

**257 prompts**, one per exercise. Visual style: **photorealistic**, consistent across every image.
Save each image as the filename shown so the website picks it up automatically.

## IMPORTANT: use a photorealistic model

The look depends mostly on the **checkpoint** you pick in ComfyUI's Load Checkpoint node.
Choose a realistic photo model or the images will look cartoonish:

- **SD 1.5:** *Realistic Vision* (use 512x512)
- **SDXL:** *RealVisXL* or *Juggernaut XL* (use 1024x1024 — sharper)

## Recommended settings

| Setting | Value |
|---|---|
| Size | 512x512 (SD 1.5) or 1024x1024 (SDXL) |
| Steps | 28 |
| CFG | 6 |
| Sampler | dpmpp_2m |
| Scheduler | karras |
